# 🧙 Aman Jaiswar | Archmage of UI

> *"Speak your spell to uncover the secrets within."*

A magical and interactive portfolio website inspired by wizardry and fantasy aesthetics. Built with modern web technologies, this project transforms a traditional portfolio into an immersive experience where visitors navigate using spells.

---

## ✨ Features

🔮 Spell-based navigation

🕯 Floating magical candles

🌌 Animated particle effects

🌙 Day/Night mode

📜 Resume summoning

📚 Projects chamber

🧠 Skills grimoire

🏆 Achievements section

✨ Glassmorphism UI

⚡ Smooth transitions and effects

---

## 🛠 Tech Stack

* HTML5
* Tailwind CSS
* JavaScript
* Google Fonts
* Material Symbols
